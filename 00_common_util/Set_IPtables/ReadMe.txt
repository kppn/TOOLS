FreeRADISUのセカンダリ試験にて

引数　説明
	"pad_set"	プライマリ認証ルートへの信号をドロップ設定する
        "pad_del"	プライマリ認証ルート・ドロップ設定を解除
        "pacd_set"	プライマリ課金ルートへの信号をドロップ設定する
        "pacd_del"	プライマリ課金ルート・ドロップ設定を解除

        "pad2_set"	プライマリ・セカンダリ　認証ルートへの信号ドロップ設定
        "pad2_del"	プライマリ・セカンダリ　認証ルート・解除
        "pacd2_set"	プライマリ・セカンダリ　課金ルートへの信号ドロップ設定
        "pacd2_del"	プライマリ・セカンダリ　課金ルート・解除
