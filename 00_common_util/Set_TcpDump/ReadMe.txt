tcpdumpの貼り付け＆出力ファイル名を容易にする
作成したファイルをFTPで指定（固定）の場所に転送する

コマンド例
	"mtd_S11"を投与する
	tcpdumpが実行され、監視可能状態になる

	"dtd_S11 660−001−001 1"を投与する
	監視と同様のtcpdumpをファイルに出力する
	ファイル名は引数によって設定される

	上記の場合だと、
	660-001-001_1_GWP000_S11.pcap

	第一引数で試験番号
	第二引数で実施ナンバー



	"ftpdtd_S11 660−001−001 1"を投与する
	上記のコマンドの場合、下記のファイルが転送される
	660-001-001_1_GWP000_S11.pcap
